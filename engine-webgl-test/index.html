Approach

Canvas APIの線画を、WebGLの小さな図形に分解。

ピストン、クランク、バルブ、配管は太い線分や矩形ポリゴンとして描きます。 粒子と背景グローは円形シェーダーで描くため、Canvas 2DよりGPUに寄せた比較ができます。

Same behavior

スクロールで回転数が上がります。

Canvas最適化版と同じように、スクロール速度からRPMを上げます。 見た目の完全一致よりも、体感負荷と再現可能性の比較を優先したプロトタイプです。

Compare

実機でCanvas版と横並び比較するための土台。

ここで明確に軽ければ、本番背景をWebGLへ寄せる価値があります。 逆に差が薄ければ、Canvas 2D最適化だけで十分かもしれません。
